# Didaktik & didaktische Modelle

## Lernziele
* Die Studierenden kennen das SAMR-Modell.
* Die Studierenden kennen die Lernziel-Taxonomie nach Bloom und können Lernziele den 6 Taxonomiestufen zuordnen.
* Die Studierenden können anhand des Pädagogischen Rades Lernanwendungen entsprechend der Taxonomiestufe auswählen.
* Die Studierenden können aus einer Modulbeschreibung die Rahmenbedingungen für eine Studienmodul ableiten.
* Die Studierenden kennen die didaktischen Modell "Constructive Alignment", das 5-Stufen-Modell von Salmon und das Instruktionsdesign ADDIE.
* Die Studierenden können E-tivities den 5 Entwicklungsstufen (Salmon) zuordnen.
* Die Studierenden kennen das Didaktische Rad und können anhand der W-Fragen (wer, was, wozu, von wem, wie, mit wem, womit, wo und wannn) die didaktischen Rahmenbedingungen für eine Kurs skizzieren.
* Die Studierenden kennen das Konzept des umgekehrten Unterrichts (Flipped / Inverted Classroom)

### Merkmale von E-Learning

--{{1}}--
Der Vorteil von E-Learning ist vorallem die Ortsunabhängigkeit. So können Lernende von unterschiedlichen Orten an einer Veranstaltung teilnehmen.

{{1-2}} * ist ortsunabhängig

--{{2}}--
Ein weiterer Vorteil ist, dass E-Learning individuelles Lernen - sowohl in bezug auf das Lerntempo als auch in bezug auf die Lernzeit - ermöglicht. E-Learning ermöglicht den Lernenden somit mehr Flexibilität beim Lernen.

{{2-3}} * ermöglicht individuelles Lernen (Lerntempo)
{{2-3}} * ermöglicht Flexibilität (Lernzeit selber einteilen)

--{{3}}--
Auf der anderen Seite erfordert E-Learning ein hohes Maß an eigenverantwortlichem Lernen, welches Motivation und Selbstorganisationsfähigkeit voraussetzt. Außerdem erfordert E-Learning ein grundsätzliches technisches Verständnis, um mit digitalen Tools effektiv umzugehen.

{{3-4}} * erfordert Selbstorganisation und eigenverantwortliches Lernen (Selbstlernkompetenz)
{{3-4}} * erfordert technisches Verständnis

--{{4}}--
Ein weiterer positiver Effekt von E-Learning ist, dass durch Online-Besprechungen Reisekosten eingespart werden können.

{{4-5}} * kann (Reise-)Kosten einsparen

### Szenarien an der Hochschule

--{{1}}--
An Hochschulen kann man zwischen drei Szenarien unterscheiden:


**1. Anreicherung von Präsenzveranstaltungen**
* Vortragsbegleitend werden digitale Fotos, Animationen und Simulationen zur Visualisierung eingesetzt.
* Für die Entwicklung von Inhalten (z. B. mathematische Gleichungen) eignen sich z. B. digitale Whiteboards mit Aufnahmefunktion.
* Abstimmungssysteme oder Twitter-Walls beziehen die Lernenden mit ein (gut geeignet für große Gruppen).

<!-- style="padding-top: 25px;" -->
**2. Blended Learning / Hybrides Lernen**

Kombination von Präsenzelementen (Workshops, Seminare, Vorlesungen) mit Online-Präsenzelementen (Lern-Video / Vorlesungsaufzeichnung, Web Based Training (WBT), Skripte, Foren, Wikis etc.)

<!-- style="padding-top: 25px;" -->
**3. Reine Online-Veranstaltung**

z. B. MOOCs, Webinare

## Bloomsche Lernziel-Taxonomie

Lernziele können unterschiedlichen Taxonomiestufen zugeordnet werden. Die Taxonomiestufen gliedern die Lernziele in hiarchische Schritte und helfen bei der Lernzielkontrolle. Den einzelnen Taxonomiestufen nach Bloom (1976) werden Verben zugeordnet, um die Lernziele zu beschreiben. In der folgenden Abbildung sehen Sie die sechs Taxonomiestufen, die über die Zeit modifiziert und manchmal auf vier Stufen reduziert werden.

<!-- style="padding-top: 25px;" -->![Image](/images/BloomscheTaxonomiestufen.jpg)<!--
style="width: 100%;" -->

<!-- style="padding-top: 25px;" -->Aus der folgenden Tabelle können Sie ablesen, welche Verben welcher Taxonomiestufe zugeordnet werden:

|     Stufe    	|     Verben    	|     Beispielsätze    	|
|:-:	|:-:	|:-:	|
|     6   - gestalten / kreieren    	|     zusammenstellen,   entwerfen, erfinden, konstruieren, planen, produzieren, herstellen      Schlüsselwörter   für Aufgaben:     wie, entwickle, was würde sein, vermute, was bedeutet, plane, stelle her   etc.    	|     Die   Studierenden können ein fiktives Modell einer neuen Art der   Signalweiterleitung entwickeln.     	|
|     5   - evaluieren    	|     auswerten,   überprüfen, vorschlagen, verallgemeinern, bewerten, beurteilen, entscheiden,   argumentieren, widerlegen, verfassen      Schlüsselwörter   für Aufgaben:      was   ist, beurteile, entscheide, kontrolliere, deute, verteidige etc.    	|     Die   Studierenden können verschiedene Arten der Signalweiterleitung bezüglich   deren Geschwindigkeit bewerten.    	|
|     4   - analysieren    	|     einstufen,   unterscheiden, gegenüberstellen, erforschen, zerlegen, auswerten, ermitteln,   ableiten, gliedern, diskutieren, differenzieren, einordnen      Schlüsselworte   für Aufgaben:     was tust du, untersuche, erkläre, vergleiche, erkenne etc.    	|     Die   Studierenden können verschiedene Arten der Signalweiterleitung bezüglich   deren Geschwindigkeit einordnen.    	|
|     3   - anwenden    	|     anwenden,   übertragen, umsetzen, ausführen, durchführen, darstellen, schildern,   benutzen, umsetzen      Schlüsselworte   für Aufgaben:     wie würdest, prüfe, betrachte, finde heraus, löse, wähle etc.    	|     Die   Studierenden können ihr Wissen über verschiedene Arten der   Signalweiterleitung auf neue Sachverhalte übertragen.    	|
|     2   - verstehen    	|     interpretieren,   erklären, darstellen, umsetzen, vergleichen, zusammenfassen, kategorisieren,   demonstrieren, übersetzen, illustrieren, beschreiben      Schlüsselworte   für Aufgaben:     was, vergleiche, ordne, gib ein Beispiel, unterscheide etc.    	|     Die   Studierenden können verschiedene Arten der Signalweiterleitung erklären.     	|
